# Icon previews

Generated previews of `ios/TokenTracker/Resources/Assets.xcassets/AppIcon.appiconset/icon-1024.png`
at the sizes iOS actually renders the icon at (with rounded-corner mask
applied to match the home-screen presentation):

- `preview-180.png` — iPhone home screen (60pt @ 3x)
- `preview-120.png` — Settings / Spotlight (40pt @ 3x or 60pt @ 2x)

These are committed so PR review can see what the icon looks like at
small sizes without spinning up a Simulator. Not loaded by the app.
