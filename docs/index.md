# TokenCounter

A privacy-friendly iOS app to monitor your Anthropic API spend.

- 💰 Live month-to-date cost across your whole Anthropic organization
- 📊 Today's intra-day cost estimate (closes the gap that Anthropic's daily report leaves open)
- 🔐 Your admin key stays in the iOS Keychain — never sent anywhere except `api.anthropic.com`
- 🚫 No analytics, no crash reporting, no account, no server
